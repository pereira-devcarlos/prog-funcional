# Estudos em Programação Funcional com Haskell

Este repositório tem como objetivo reunir anotações, exemplos e exercícios relacionados ao estudo de Programação Funcional utilizando a linguagem Haskell.

## Sobre

Haskell é uma linguagem de programação puramente funcional, conhecida por seu forte sistema de tipos, imutabilidade e suporte a funções de alta ordem. Este repositório serve como material de apoio para quem deseja aprender ou aprofundar seus conhecimentos em Haskell e conceitos de programação funcional.

## Estrutura

- Exemplos de código comentados
- Exercícios resolvidos e propostos
- Anotações teóricas sobre conceitos fundamentais (funções puras, recursão, composição, monads, etc.)
- Referências e links úteis

## Como usar

1. Clone este repositório:
   ```sh
   git clone https://github.com/seu-usuario/funcional.git
   ```
2. Navegue pelos arquivos e pastas para acessar os exemplos e exercícios.
3. Para executar os exemplos, é necessário ter o [GHC](https://www.haskell.org/ghc/) instalado.
